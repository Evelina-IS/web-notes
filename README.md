# Web Notes

A local-first Markdown notes app with preview, handwriting, PDF references, and GitHub-backed sync.

Open `index.html` in a browser to use it.

## GitHub Sync

Use **同步设置** in the app and fill in:

- Owner
- Repo
- Path, for example `notes.json`
- A fine-grained GitHub token with repository Contents read/write permission

When automatic sync is enabled, the app pulls on startup and pushes local edits after a short delay.
